# AGENTS.md

## Mission
Rebuild the RAMSLAB website from scratch so that its visual language, information architecture, layout rhythm, and editorial feel are as close as possible to https://senseable.mit.edu/ while using only RAMSLAB content and assets.

## Priority
The result must feel structurally and visually very close to the reference website.
Do not invent a new visual identity.
Do not reinterpret creatively.
Creative freedom is minimal.

## Hard constraints
- Do NOT create a warm-toned site.
- Do NOT use ochre, beige, terracotta, mustard, brown, cream, or editorial sepia palettes.
- Do NOT build a long one-page scrolling landing page.
- Do NOT use startup/SaaS aesthetics.
- Do NOT use gradients, blobs, glassmorphism, frosted effects, or decorative animations.
- Do NOT use oversized rounded cards.
- Do NOT use playful branding elements.
- Use a strict black / white / neutral gray palette.
- The site must feel editorial, institutional, minimal, research-oriented, and architecture-like.

## Visual direction
- White background
- Black or near-black typography
- Neutral grays only
- Minimal accent color, only if strictly necessary
- Clean grid
- Strong hierarchy through typography and whitespace
- Dense but elegant editorial rhythm
- Large media blocks where needed
- Strong list/detail page structure
