# RAMSLAB Design Tokens (Figma Reference)

This file contains the core design tokens extracted from the CSS to help you set up Styles in Figma.

## 🎨 Colors
Use these for Color Styles.

| Token | Hex Value | Usage |
| :--- | :--- | :--- |
| `--bg` | `#FFFFFF` | Main Background |
| `--text` | `#111111` | Primary Headings & Text |
| `--text-soft` | `#2B2B2B` | Body Copy (Serif) |
| `--muted` | `#5F5F5F` | Captions, Subtitles, Meta |
| `--muted-soft` | `#7D7D7D` | Secondary Meta |
| `--line` | `#D8D8D8` | Section Borders, Dividers |
| `--line-soft` | `#EBEBEB` | Item Dividers |
| `--surface` | `#F2F2F2` | Placeholder Backgrounds |
| `--surface-strong` | `#E5E5E5` | Placeholder Accents |

## Typography
Use these for Text Styles.

| Type | Font Family | Size | Weight | Line Height |
| :--- | :--- | :--- | :--- | :--- |
| **Page Heading** | Sans-serif | 38px - 80px | 700 | 0.96 |
| **Section Title** | Sans-serif | 12px (0.75rem) | 700 | 1.0 (Caps) |
| **Body (Serif)** | Serif | 16px (1rem) | 400 | 1.8 |
| **Archive Title** | Sans-serif | 18px (1.12rem) | 700 | 1.14 |
| **Meta / Label** | Sans-serif | 12px (0.75rem) | 700 | 1.0 (Caps) |
| **Caption** | Sans-serif | 13px (0.82rem) | 400 | 1.5 |

*Note: Sans-serif is "Helvetica Neue" / Arial. Serif is Georgia / Times New Roman.*
